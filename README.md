# 👗 Wardrobe - 衣橱扩展

SillyTavern 角色穿搭生成扩展。通过 AI 为角色生成创意服装描述，自动注入聊天上下文，让 AI 感知角色当前穿着。

## 功能

- **AI 生成穿搭**：基于标签约束（类型/风格/颜色/材质）调用 LLM 生成服装描述
- **多角色支持**：可同时为多个角色设定穿搭，全部注入 AI 上下文
- **标签选择系统**：覆盖男/女各 9 大品类（上衣、下装、内衣、袜子、鞋、头饰、首饰、背包、整套），每个品类有多个维度可选
- **全局指导**：输入场景/氛围关键词（如"疫情时期"、"仙侠世界"），AI 会据此调整生成风格
- **人物设定注入**：填写角色性格/人设摘要，AI 会根据角色特点匹配穿搭
- **穿搭收藏**：保存常用搭配，一键加载到现有穿搭
- **双 API 模式**：可复用酒馆主模型（零配置），也可配置独立的 OpenAI 兼容 API（省钱）
- **轻量注入**：不修改原始角色卡，通过 `setExtensionPrompt` 注入，token 消耗极少

## 安装

### 方式一：酒馆内安装（推荐）

1. 打开 SillyTavern
2. 点击扩展管理（🧩 图标）
3. 点击「安装扩展」
4. 输入本仓库地址：`https://github.com/Jovial020329/SillyTavern-Wardrobe`
5. 安装完成后重启酒馆

### 方式二：手动安装

1. 下载本仓库所有文件
2. 放入 `SillyTavern/data/default-user/extensions/wardrobe/` 目录
3. 重启 SillyTavern

## 使用方法

1. 打开扩展面板（🧩 图标），找到「👗 衣橱 Wardrobe」，点击展开
2. 选择性别（♀ 女 / ♂ 男）
3. （可选）展开「全局指导」填写场景背景
4. （可选）展开「人物设定注入」填写角色性格
5. 点击分类标签页，选择偏好标签（不选则完全随机）
6. 在「角色名」输入框填写角色名（如：小雪）
7. 点击「🎲 随机生成」→ AI 生成穿搭描述
8. 点击「✅ 应用」→ 穿搭加入「现有穿搭」列表，自动注入 AI
9. 重复以上步骤可为多个角色添加穿搭
10. 点击角色旁的 ✕ 可移除该角色穿搭

## 标签编辑

仓库内附带 `标签编辑器.html` 工具（不包含在扩展安装中），可在浏览器中打开，可视化编辑服装标签分类，支持：

- 添加/删除标签
- 添加/删除子维度（如在「上衣」下新增「品牌」维度）
- 添加/删除整个分类
- 编辑完导出新的 `wardrobe-categories.js` 上传覆盖即可

## 文件结构

```
wardrobe/
├── manifest.json          # 扩展清单
├── index.js               # 主逻辑
├── style.css              # 样式
└── wardrobe-categories.js # 服装分类标签数据
```

## 注意事项

- 生成穿搭需要已配置好酒馆的 API（或在扩展内配置独立 API）
- 注入的穿搭描述每轮对话都会自动发送给 AI，直到手动移除
- 不会修改原始角色卡数据
